## 📋 What is 1nuo-rate?

1nuo-rate is a tool designed for bloggers who want to rate content and manage reviews without hassle. It works with many static blog platforms like Hexo, Hugo, and Jekyll. Instead of using a server, it updates data through GitHub's API. This means you don’t need to run or pay for any backend service.

The ratings and reviews show up in clear, visual charts powered by ECharts. You can edit your data using a simple interface that saves as JSON files on GitHub. This lets you keep your blog content and reviews tidy and easy to update.
